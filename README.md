# MD16 Computer Academy

React + Vite frontend for the MD16 Computer Academy landing page, built with React Bootstrap.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Main Files

- `src/App.jsx` - page layout and content
- `src/App.css` - custom styling
- `src/main.jsx` - app entry and Bootstrap imports

## Notes

- The contact form is frontend-only.
- External links use WhatsApp, phone, map, and social links from the page content.
